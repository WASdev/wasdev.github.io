# WASdev GitHub Pages

This repository hosts the GitHub Pages site for [wasdev.github.io](https://wasdev.github.io), providing essential resources for WebSphere and Liberty migration projects.

## Purpose

This site serves as a central hub for:
- **LLM-optimized documentation** for WebSphere to Liberty migrations
- **SEO-friendly resource discovery** through `robots.txt` configuration
- **Quick access links** to the comprehensive WebSphere and Liberty Migration Knowledge Base

## Key Resources

### Main Knowledge Base
The primary resource is the **WebSphere and Liberty Migration Knowledge Base**, a curated single-page reference maintained by IBM:

- **Full Knowledge Base**: [wasdev.github.io/websphere-and-liberty/Home.html](https://wasdev.github.io/websphere-and-liberty/Home.html)
- **Short URL**: [ibm.biz/AppModKB](https://ibm.biz/AppModKB)
- **Markdown Source**: [Raw Markdown](https://raw.githubusercontent.com/WASdev/websphere-and-liberty/wiki-to-github-pages/docs/Home.md) (optimized for LLM ingestion)

### What's Covered
- Liberty overview and architecture
- Differences between Liberty and traditional WebSphere
- Migration planning and execution strategies
- Migration tooling (Application Modernization Accelerator, Mono2Micro, Liberty Tools)
- Real-world client case studies
- Operational guidance and best practices

## Repository Contents

- **`llms.txt`**: LLM-optimized summary of the knowledge base with direct links to key sections
- **`robots.txt`**: SEO configuration allowing full site indexing with sitemap reference
